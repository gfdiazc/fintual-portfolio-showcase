# Preguntas Estratégicas para Notebook LLM

## Contexto para Notebook LM

Antes de implementar, necesitamos alinear el proyecto con la filosofía, producto y arquitectura de Fintual. Estas preguntas ayudarán a extraer insights de la documentación interna de Fintual.

---
## Workflow con Notebook LM

### Paso 1: Preparar Context para Notebook LM

Sube los siguientes documentos a tu Notebook:
1. Toda la documentación de Fintual que hayas recopilado
2. Este plan de arquitectura
3. La especificación de API de Fintual (del swagger.json)
4. Job description completa
5. Cualquier blog post, artículo técnico de Fintual

### Paso 2: Ejecutar Prompts por Categoría

Ejecuta los prompts de cada categoría (1-6) y documenta respuestas en:
`docs/notebook_lm_research/`
├── `01_product_philosophy.md`
├── `02_technical_architecture.md`
├── `03_investment_features.md`
